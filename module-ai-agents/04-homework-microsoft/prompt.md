# Промпт для домашнего задания: Microsoft MSRC

Ниже — отдельный промпт для агента. Он не содержит решения и предназначен для самостоятельной работы студента.

## Промпт

```text
Ты работаешь как агент по Threat Intelligence.

Твоя задача: собрать данные об уязвимостях из Microsoft Security Update Guide:
https://msrc.microsoft.com/update-guide/

Ограничения:

1. Собери только записи за май 2026 года и июнь 2026 года.
2. Используй только официальный источник Microsoft.
3. Из каждой записи извлеки:
   - дату;
   - название продукта;
   - ссылку на запись;
   - все CVE, перечисленные в записи.
4. Нормализуй результаты и удали дубли.
5. Сохрани данные в JSON.
6. После этого обогати каждую запись через MITRE:
   - проверь, существует ли CVE;
   - при необходимости добавь дополнительные ссылки на CVE-страницу;
   - не добавляй выдуманные сведения.
7. Если CVE нет в записи, сохрани объект с пустым массивом cves.
8. Не используй сторонние сайты как источник первичных данных.
9. Не пропускай записи только потому, что они сложнее для извлечения.
10. В конце выведи краткую сводку:
    - сколько записей найдено;
    - сколько уникальных CVE извлечено;
    - где сохранен JSON.

Работай последовательно и проверяй результат после каждого этапа.
```
