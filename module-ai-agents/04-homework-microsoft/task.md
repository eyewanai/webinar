# Домашнее задание: собрать CVE из Microsoft MSRC за май и июнь 2026 года

Источник задания:

- [Microsoft Security Update Guide](https://msrc.microsoft.com/update-guide/)

## Формулировка

Соберите все данные по уязвимостям из Microsoft Security Update Guide за:

- май 2026;
- июнь 2026.

Результат сохраните в JSON.

## Требования

- использовать только официальный MSRC Update Guide;
- собрать записи только за май и июнь 2026 года;
- сохранить результат в структурированном JSON;
- после извлечения обогатить данные через MITRE;
- не давать ручной список вместо автоматического сбора;
- не пропускать повторяющиеся записи без проверки дублей.

## Что нужно учитывать

- у записи может быть несколько CVE;
- у одной уязвимости может быть несколько ссылок;
- некоторые записи могут быть связаны с несколькими продуктами;
- данные нужно нормализовать, чтобы их можно было использовать в дальнейшем анализе.

## Практические команды

Проверка источника:

```bash
curl https://msrc.microsoft.com/update-guide/
```

Проверка локального JSON после сохранения:

```bash
python -m json.tool microsoft-msrc-2026-05-06.json
```
