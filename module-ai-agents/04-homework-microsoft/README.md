# Домашнее задание: Microsoft MSRC Update Guide

Цель домашнего задания: повторить схему из практики Apple на другом официальном источнике.

Ссылки:

- [Microsoft Security Update Guide](https://msrc.microsoft.com/update-guide/)

## Что будет отрабатывать участник

- поиск данных за конкретные месяцы;
- извлечение CVE из бюллетеней;
- сохранение результатов в JSON;
- последующее обогащение через MITRE;
- проверку дублей и качества данных.

## Практические команды

Проверка доступности источника:

```bash
curl -I https://msrc.microsoft.com/update-guide/
```

Ожидаемый результат: сервер Microsoft отвечает HTTP-заголовками.

Проверка даты в рабочей среде:

```bash
date
```

Ожидаемый результат: терминал выводит текущую дату. Это помогает не перепутать месяцы и год в задании.
