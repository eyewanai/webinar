# Промпт для агента

Промпт ниже задает агенту последовательность действий для практики.

Defanged IOC — безопасная запись индикатора, например `example[.]com`. Enrichment — обогащение IOC данными из внешнего источника.

```text
Используй Playwright MCP, чтобы открыть статью:
https://thedfirreport.com/2023/05/22/icedid-macro-ends-in-nokoyawa-ransomware

Получить текст страницы.

Используй MCP server ti-tools:

1. extract_iocs_from_text(text)
2. enrich_iocs(iocs)
3. save_iocs_to_json("ioc.json", data)

Требования:
- извлеки domains и ipv4;
- объедини их в один список;
- не теряй defanged IOC;
- enrichment делай через RST Cloud API;
- сохрани результат в ioc.json;
- в финальном ответе покажи:
  - сколько IOC найдено;
  - сколько успешно обогащено;
  - путь к JSON;
  - 3 примера записей.
```

## Как читать этот промпт

Промпт короткий и прямой. Он ведет агента по шагам:

- открыть страницу;
- получить текст;
- извлечь IOC;
- обогатить IOC;
- сохранить JSON;
- показать результат.

Такой формат снижает риск, что агент начнет искать лишние источники или пропустит сохранение JSON.

## Практические команды

Проверить, что файл промпта доступен:

```bash
cat module-ai-agents/06-custom-ti-mcp/prompt.md
```

Ожидаемый результат: терминал выводит текст промпта.

Проверить JSON после выполнения:

```bash
jq '.[0]' ioc.json
```

Ожидаемый результат: терминал выводит первую запись из JSON-массива.

## Проверка результата

- промпт понятен без дополнительных пояснений;
- в нем есть и Playwright, и `ti-tools`;
- агент понимает, что сначала нужен текст, а потом enrichment;
- итог сохраняется в `ioc.json`.
