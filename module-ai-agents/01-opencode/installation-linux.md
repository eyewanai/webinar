# Установка OpenCode на Linux

На Linux у OpenCode обычно меньше сюрпризов, чем на Windows, но больше вариантов установки. Для учебного модуля лучше выбирать путь, который легко повторить: Node.js + npm или официальный install script.

## Что понадобится

- Linux-дистрибутив;
- терминал;
- Node.js;
- npm;
- доступ в интернет;
- права на установку глобальных пакетов.

Официальные ссылки:

- [OpenCode Intro](https://opencode.ai/docs/)
- [OpenCode Download](https://opencode.ai/download)
- [Node.js Downloads](https://nodejs.org/en/download)
- [npm Documentation](https://docs.npmjs.com/)

## Вариант 1. Установка через npm

Сначала проверьте наличие Node.js:

```bash
node -v
npm -v
```

Если Node.js нет, установите его через официальный дистрибутив для вашего Linux или через системный пакетный менеджер, если он предоставляет актуальную версию.

Затем установите OpenCode:

```bash
npm install -g opencode-ai
```

Почему это хороший учебный вариант:

- не зависит от конкретного дистрибутива;
- легко копируется между машинами;
- удобно обновляется той же командой.

## Вариант 2. Установка через официальный install script

OpenCode также предлагает официальный install script:

```bash
curl -fsSL https://opencode.ai/install | bash
```

Когда этот вариант удобен:

- если вы хотите быстро поднять среду;
- если у вас уже настроена политика на установку из официального скрипта;
- если вы работаете в учебной песочнице.

## Вариант 3. Установка через системный пакетный менеджер

В официальной документации OpenCode отдельно упоминаются Linux-варианты для Arch Linux:

```bash
sudo pacman -S opencode
paru -S opencode-bin
```

Используйте этот путь только если ваш дистрибутив и учебная среда действительно поддерживают такой сценарий.

## Шаги проверки

После установки проверьте:

```bash
opencode --help
opencode
```

Если OpenCode запускается, значит базовая установка прошла успешно.

## Обновление

Для npm-установки обновление такое же, как повторная глобальная установка:

```bash
npm install -g opencode-ai
```

Для script-based установки обновление обычно выполняется повторным запуском официального install script.

## Практические команды

```bash
node -v
npm -v
npm install -g opencode-ai
opencode --help
curl -fsSL https://opencode.ai/install | bash
```

## Что должно получиться

- Node.js и npm доступны в системе;
- OpenCode установлен и запускается;
- вы понимаете, какой способ установки выбран в вашей лаборатории;

## Проверка результата

- проверьте версию Node.js и npm;
- выполните `opencode --help`;
- если используете Arch Linux, убедитесь, что пакет ставится через тот менеджер, который принят в вашей среде;
- при ошибках сверьтесь с официальной страницей загрузок.
