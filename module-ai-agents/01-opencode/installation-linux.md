# Установка OpenCode на Linux

Цель: установить OpenCode на Linux и проверить запуск из терминала. Для повторяемости обычно выбирают Node.js + npm или официальный install script.

## Что понадобится

- Linux-дистрибутив;
- терминал;
- Node.js;
- npm;
- доступ в интернет;
- права на установку глобальных пакетов.

Ссылки:

- [OpenCode Intro](https://opencode.ai/docs/)
- [OpenCode Download](https://opencode.ai/download)
- [Node.js Downloads](https://nodejs.org/en/download)
- [npm Documentation](https://docs.npmjs.com/)

## Вариант 1. Установка через npm

Сначала проверьте наличие Node.js:

```bash
node -v
npm -v
```

Ожидаемый результат: терминал выводит версии Node.js и npm.

Если Node.js нет, установите его через официальный дистрибутив для вашего Linux или через системный пакетный менеджер, если он предоставляет актуальную версию.

Затем установите OpenCode:

```bash
npm install -g opencode-ai
```

Ожидаемый результат: npm устанавливает OpenCode без ошибок.

Почему этот вариант подходит:

- не зависит от конкретного дистрибутива;
- легко копируется между машинами;
- обновляется той же командой.

## Вариант 2. Установка через официальный install script

OpenCode также предлагает официальный install script:

```bash
curl -fsSL https://opencode.ai/install | bash
```

Ожидаемый результат: скрипт скачивает и устанавливает OpenCode.

Этот вариант подходит, если:

- вы хотите быстро поднять среду;
- у вас разрешена установка из официального скрипта;
- вы работаете в изолированном окружении.

## Вариант 3. Установка через системный пакетный менеджер

В официальной документации OpenCode отдельно упоминаются Linux-варианты для Arch Linux:

```bash
sudo pacman -S opencode
paru -S opencode-bin
```

Ожидаемый результат: пакетный менеджер устанавливает OpenCode.

Используйте этот путь только если ваш дистрибутив и среда действительно поддерживают такой сценарий.

## Шаги проверки

После установки проверьте:

```bash
opencode --help
opencode
```

Ожидаемый результат: первая команда выводит справку, вторая запускает OpenCode.

## Обновление

Для npm-установки обновление такое же, как повторная глобальная установка:

```bash
npm install -g opencode-ai
```

Ожидаемый результат: npm обновляет установленный пакет или сообщает, что актуальная версия уже установлена.

Для script-based установки обновление обычно выполняется повторным запуском официального install script.

## Критерий завершения

Установка завершена, если `opencode --help` выводит справку, а команда `opencode` запускает клиент.
