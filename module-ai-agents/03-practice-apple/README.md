# Практика: Apple Security Updates и извлечение CVE

Цель практики: с помощью агента пройти по официальной странице Apple, собрать macOS CVE за 2026 год и сохранить результат в JSON.

Ссылки:

- [Apple Security Releases](https://support.apple.com/en-us/100100)

## Зачем это нужно

В реальной работе данные редко приходят в готовом виде. Чаще приходится:

- открыть страницу производителя;
- понять, как устроена навигация;
- найти релевантные записи;
- извлечь идентификаторы CVE;
- сохранить это в формате, пригодном для дальнейшего анализа.

Агент с браузерной автоматизацией помогает пройти эти шаги последовательно.

## Что вы изучите

- как формулировать задачу агенту;
- как агент навигирует по странице;
- как он извлекает CVE из текста;
- как он использует регулярное выражение для поиска CVE;
- как обогащает запись дополнительными данными;
- как сохраняет структуру в JSON.

## Практические команды

Проверка доступа к странице:

```bash
curl -I https://support.apple.com/en-us/100100
```

Ожидаемый результат: сервер Apple отвечает HTTP-заголовками, например со статусом `200` или перенаправлением.

Проверка окружения:

```bash
node -v
npm -v
```

Ожидаемый результат: терминал выводит версии Node.js и npm.

## Критерий завершения

Практика завершена, если JSON-файл создан, проходит проверку синтаксиса и содержит только macOS-записи за 2026 год из официального источника Apple.
