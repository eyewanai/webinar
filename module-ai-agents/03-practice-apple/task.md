# Практическая задача: собрать все macOS CVE за 2026 год

Задача: настроить агентный сценарий, который проходит по Apple Security Releases и собирает записи macOS за 2026 год.

Ссылки:

- [Apple Security Releases](https://support.apple.com/en-us/100100)

## Цель

Получить полный список macOS CVE за 2026 год из официальных бюллетеней Apple и сохранить результат в JSON.

## Ожидаемый результат

На диске должен появиться JSON-файл со списком записей. В каждой записи есть продукт, ссылка на бюллетень, уникальный идентификатор и массив CVE.

## Что именно нужно собрать

- все записи, относящиеся к macOS;
- все CVE, связанные с этими записями;
- название продукта;
- ссылку на страницу бюллетеня;
- уникальный идентификатор записи;
- результат в структурированном JSON.

## Ограничения

- использовать только официальный источник Apple;
- не подменять данные другими сайтами;
- не собирать iOS, watchOS, tvOS, visionOS, Safari и другие категории, если они не относятся к macOS;
- не собирать данные за другие годы;
- не писать ручной список вместо автоматического извлечения.

## Что нужно предусмотреть

- переход по страницам;
- поиск нужных строк;
- извлечение CVE через регулярные выражения;
- удаление дублей;
- сохранение JSON на диск.

## Практические команды

Проверка источника:

```bash
curl https://support.apple.com/en-us/100100
```

Ожидаемый результат: команда выводит HTML-страницу или сообщает о проблеме доступа к источнику.

Проверка даты и года в задаче:

```bash
date
```

Ожидаемый результат: терминал выводит текущую дату, чтобы вы не перепутали год в фильтре.

## Критерий завершения

Задание готово, если результат содержит только macOS за 2026 год, CVE не продублированы внутри записей, а JSON проходит проверку валидности.
