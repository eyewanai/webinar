1. `tor_connection.py` парсит `.onion`-ссылки.

Перед запуском подключитесь к Tor Browser, чтобы прокси был активен.

Если сайт показывает captcha или DDoS-проверку, пройдите ее вручную в браузере. Затем через Chrome DevTools во вкладке Network возьмите cookies и передайте их в `requests.get()`.

Для браузерной автоматизации можно использовать Selenium: он открывает браузер и автоматизирует клики, ввод текста и переходы.
https://selenium-python.readthedocs.io

2. `parse_ddos_attacks` парсит GitHub-страницу с потенциальными DDoS-атаками.

3. `telegram_parse.py` парсит выбранный Telegram-канал.

Перед запуском установите Telethon:

```bash
pip3 install Telethon==1.31.0
```

Ожидаемый результат: pip устанавливает библиотеку без ошибок.

Затем получите `api_id` и `api_hash` на https://my.telegram.org/auth?to=apps и ID канала. ID канала должен начинаться с `-100`.

Telegram ограничивает частые запросы, поэтому используйте большой таймаут или несколько аккаунтов.
